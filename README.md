# Token Tycoon

Token Tycoon is a small browser clicker game built with plain HTML, CSS, and JavaScript.
Click the character to earn points, see a quick visual feedback, and keep progress in `localStorage`.

## Features

- Click-to-earn score mechanic (`+1` per click)
- Click tilt animation and floating `+1` feedback
- Image switch after reaching `50` points
- Score persistence in browser `localStorage`

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- No backend, no bundler, no external dependencies
